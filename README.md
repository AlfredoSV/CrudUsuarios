## Comenzando 🚀

*Clonar el proyecto

*Cambiar la cadena de conexión que se encuentra en appsettings.json

*Ejecuntar el script.sql en su sistema gestor de bases de datos


### Pre-requisitos 📋

*El SDK .Net Core 3.1

*Sql Server > 2014
